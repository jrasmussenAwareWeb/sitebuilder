Usage

Use Navbar Utility Nav for secondary navigation menus with user defined links.

Tips & Troubleshooting

Linking to a Modal

A link to a modal window requires additional attributes that are not available within a Navbar Menu Item. Use a Rich Text Editor or HTML component with the below sample code. 

<a href="#" data-toggle="modal" data-target="#ModalWindowLogin">Login</a>
