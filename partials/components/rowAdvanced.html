Usage

Use Row Advanced to create horizontal groups of columns when you need more flexibility on column size options for breakpoints.
Selection options are available for Multiple Breakpoints, Single Breakpoint - sm, and Single Breakpoint - xs as listed in Content Group Options.
Additional column classes can also be added. See Adding Column Sizes for more information.

When you need only one breakpoint for mobile ( < 768px ), use Row.
