Scroll to Top

Allowed In

  • Container
  • Row
  • Row Advanced

Usage

Use to provide a floating navigation arrow for users to return to the top on longer pages.

Examples

Web

Code Sample


<a href="#scrollNav" style="float: right;" data-scroll="" data-spy="affix" data-offset-top="100" data-offset-bottom="20" class="btn btn-primary scrolltop affix" title="top">
    <i class="fa fa-chevron-up"></i>
</a>

Tips & Troubleshooting

Placement

Wherever you place the Scroll To Top, it will update to be in the lower right corner.
Recommendation is to place it in its own row below any content area in the body section, either in a base layout or on a page.


Overwriting Colors

To overwrite the default colors, you will have to target the btn-primary for the Scroll To Top.
Add the following code to the custom stylesheet and update the colors.

a.btn.btn-primary.scrolltop {
    background-color: #0785a4;
    border-color: #06718c;
    color: #fff;
}
