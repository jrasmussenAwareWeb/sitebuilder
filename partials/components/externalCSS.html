External CSS Reference

Allowed In

  • Headscripts & CSS

Usage

Use External CSS Reference to link to external stylesheets — for example Google Fonts, Font Awesome, etc.

Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
External CSS URL Input Enter the external CSS URL reference — example: 'https://fonts.googleapis.com/css?family=Raleway'
Media Input Enter media type — example(s): 'print', 'all', 'screen'
