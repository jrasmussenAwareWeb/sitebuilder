Utility Navigation

Allowed In

  • Container
  • Modal
  • Row
  • Row Advanced
  • Utility Navigation

Usage

Use Utility Navigation for secondary navigation menus with user defined links.

Example

Content Group Options


None

Allowed Components

  • Navbar Menu Item
  • Utility Navigation

Code Sample

<ul class="nav navbar-nav sm" style="" data-smartmenus-id="">
    <li><a href="/GenericSampleCompany/About" title="About">About</a> </li>
    <li><a href="/GenericSampleCompany/ResourceCenter" title="Resources">Resources</a> </li>
</ul>

Classes

.nav > li {
    display: block;
    position: relative;
}
.navbar-nav {
    float: left;
    margin: 0;
}
.navbar-nav > li {
    float: left;
}
navbar-nav > li > a {
    padding-bottom: 20px;
    padding-top: 20px;
    line-height: 20px;
}
