Social Connect

Allowed In

  • Accordion
  • Container
  • Footer
  • Row
  • Row Advanced

Usage

Use Social Connect to provide links to specific social media applications — for example when you want to link to your organizations facebook and twitter pages, you might put them in the header or footer areas.

Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Title Input Enter a title header (optional)
Vertical Orientation Checkbox Display links vertically
Hide All Labels Checkbox Hide all Link Description labels
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)

Add Icon


To add a social media icon, click the Insert a new item icon .

Sort Icons


To reorder icons, click the Sort icon .

Add Icon Link

Once you add an icon, you must also add a link.


To add an icon link, click the Edits the link destination icon.

Icon Link Settings
  1. Click External Link
  2. Enter Link Description — this text renders to the right of the icon (optional)

    To hide the link description, check the Hide All Labels checkbox in Social Connect options.

  3. Enter the URL for social media application (specific company) page
  4. Set Target Window to 'New Browser' to open the link in a new tab

Content Group Options - Icon


To edit content group options, click the Edit Content Group icon .

Reorder Icon Links


To reorder icons, click the up or down arrow icons .

Delete Icon Link


To delete an icon link, click the Clears the Link icon .
