Navbar Logo with Text

We should add a surrounding element (<span>) ? to the text so it can more easily be targeted for styling

Potentially update CSS to account for text that is not linked - see Tips & Troubleshooting

Allowed In

  • Navbar Alternate

Usage

Use Navbar Logo with Text for an image with text to the right — text can optionally be linked.

Examples

Mobile (stacks)

Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Image Media Library Set background image from media library
(Image) Link Content Selector
Logo Height Input Add 20 to desired logo height to account for padding (e.g. for 80 pixel logo enter '100px')
Text Input Enter text — will display to the right of the image (optionally link)
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)

Tips & Troubleshooting

If not linking text

If the text is not linked the image will be larger than if the text is linked because the .navbar-brand class is not wrapping the image.

Linked

Not Linked

Add correction below to the Custom Stylesheet. 
.navbar-header > img {
    display: inline-block;
    max-height: 100%;
    max-width: 100%;
    width: auto;
}

Mobile Correction
Add below correction to the Custom Stylesheet to keep the text from flowing over the content container. 
@media(max-width:767px) {
    .navbar-brand {
        padding:0 15px;
    }
}
