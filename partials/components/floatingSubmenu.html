Floating Submenu

Allowed In

  • Row
  • Row Advanced

Usage

Use Floating Submenu for anchor link navigation on a single page — for example, a really long page with multiple sections. Clicking the navigation item will take the user to the corresponding section on the page.


More information at getbootstrap.com
http://getbootstrap.com/components/#nav

  • None

CSS Class Input Add CSS class
Style Input Add CSS styles (inline)

Add Floating Submenu Top Level Items

To insert a Floating Submenu Item at the top (parent) level , you must do so from the Floating Submenu —

  1. Using the content tree dropdown, navigate from Floating Submenu Item to Floating Submenu



  2. click the Insert a new item icon

Content Group Options — Floating Submenu Item (Label)


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Jump to CSS ID Input Enter the CSS ID for the corresponding page section to jump to — enter this ID in the CSS ID field of the component, i.e. Row, Title, etc.
Label Input Enter link text
CSS ID Input Enter ID
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)

Add Floating Submenu Child Items

To insert children of the top (parent) level, do so from the Floating Submenu Item.

Floating Submenu Item Ordering


To reorder Floating Submenu Items, click the up or down arrow icons .

Floating Submenu Item Delete


To delete a Floating Submenu Item, click the Delete Row icon .

Tips & Troubleshooting

Because Floating Submenu is specific to a page, its not recommended on a base layout.
