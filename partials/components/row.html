Usage

Use rows to create horizontal groups of columns when simple breakpoints will work with your design.
Selection options are available for One Column, Two Column, Three Column, and Four Column rows as listed in Content Group Options.

For each Row Definition selection a class is added to each column of .col-sm-*.
This means that all columns will have a class of .col-xs-12 .col-sm-* resulting in one breakpoint for mobile ( < 768px ).
When you need more flexibility on column size options for breakpoints, use Row Advanced.
