Jumbotron

Allowed In

  • Accordion
  • Container
  • Row
  • Row Advanced
  • Tabset

Usage

Use for a single feature large callout — includes a background image, title, body text, and optional button link. To feature multiple items, use Billboard or Carousel.


More information at getboostrap.com
http://getbootstrap.com/components/#jumbotron

Tips & Troubleshooting

Restrictions & Dependencies

The Jumbotron component is intended for a single feature item. To feature multiple items, use Billboard or Carousel.
