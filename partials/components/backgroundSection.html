Background Section

Allowed In

  • Header
  • Body
  • Footer

Usage

Use a Background Section when you need a full-width or fixed-width section background image or background color.
Add a Container within the Background Section for additional content.

Cover scales the background image to be as large as possible so that the background area is completely covered by the background image.
Some parts of the background image may not be in view within the background positioning area.

Bootstrap & Other Classes

The Background Section does not use specific Bootstrap classes.
The below example is a result of the Content Group Option selections and an additional min-height style set via the Content Group Options Style input.

background-image: url("/-/media/[file_path].ashx");
background-position: left top;
background-repeat: no-repeat;
background-size: 100% 440px;
min-height: 440px;

Tips & Troubleshooting

If a Background Section image is not showing or only partially showing, most likely there is not a height/width set.
Set via the following methods:

  • add a class to the Background Section and declare via CSS
  • add a class to the Background Section child Container and declare via CSS
  • set a height/width in the Background Section height/width in the Content Group Option selections and include a height/min-height or width/min-width in Content Group Options Style input
