Navbar Font Resizer

Allowed In

  • Navbar
  • Navbar Jump Menu

Usage

Provide the ability for the user to increase or decrease the font size of the page or site. Increments or decrements font size by 1px per click.

Examples

Site Builder

Web

Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Alignment Selection Select navbar alignment using default class options
  • Left — .navbar-left
  • Right — .navbar-right
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)
