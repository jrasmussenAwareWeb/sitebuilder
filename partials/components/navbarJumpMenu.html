Usage

Use Navbar Jump Menu for a fixed — (does not scroll, other content scrolls under) — top header with options for Search, Navbar Logo, Navbar Font Resizer, Navbar Google Translator, Navbar Menu Item(s), and Navbar Utility Nav.

When placing additional components within the Navbar Jump component and checking the Include Search Box checkbox, the Search icon will always float to the left of any placed components that are set to align right.
