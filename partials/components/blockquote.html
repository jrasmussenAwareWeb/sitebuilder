Blockquote

Allowed In

  • Accordion
  • Billboard Slide
  • Container
  • Row
  • Row Advanced

Includes

  • Text
  • Footer
  • Source Text

Usage

Use to apply default bootstrap blockquote styles for a section that is quoted from another source.


More information at getbootstrap.com
http://getbootstrap.com/css/#type-blockquotes

Code Sample

/**default**/
<blockquote class="blockquote" style="">
    <p>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla sit amet velit vel felis dictum sodales.
    Donec volutpat ultrices felis sit amet tincidunt. Nullam sed auctor augue.
    </p>
    <footer>
        Satisfied Kitten Litter Owners<i></i>
    </footer>
</blockquote>
/**align right**/
<blockquote class="blockquote-reverse" style="">
    <p>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla sit amet velit vel felis dictum sodales.

Bootstrap Classes

.blockquote {
    border-left: 5px solid #eee;
    font-size: 17.5px;
    margin: 0 0 20px;
    padding: 10px 20px;
}
.blockquote-reverse,
blockquote.pull-right {
    border-left: 0 none;
    border-right: 5px solid #eee;
    padding-left: 0;
    padding-right: 15px;
    text-align: right;
}
