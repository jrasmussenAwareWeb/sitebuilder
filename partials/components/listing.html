Content Group Options — Listing


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Start Item Content Selector
Search All Descendents Checkbox
Tags Selection (multiple)
Max Items To Display Input Set to 0 to return all results
Sort Order Selection Select the sort order using default options
  • Alphabetical
  • Content Tree Order
  • Newest
  • Oldest
  • Recently Updated
Display Images Checkbox
Display Summaries Checkbox
Max Summary Characters Input
Link Text Input Enter Link Text for all listing items
CSS ID Input Add ID
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)

If Link Text is set here, it will override any input entered in the Listing Item content group options.

Content Group Options — Listing Item


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Listing Image Media Library
Abstract Text Area
Link Text Input Enter Link Text for individual listing item

If Link Text is set in the Listing content group options (parent), it will override any input here.

Image Select

To select an image, click the Choose an Image icon.

Image Edit

To edit an image, click the Modify Image Appearance icon.


Image Properties
Option Data Type Function
Text Input Alternate Text
Width Input Set the width of the image if they are different than the dimensions of the original
Height Input Set the height of the image if they are different than the dimensions of the original
Keep Aspect Ratio Checkbox
Horizontal space Input Set the horizontal space around the image
Vertical Space Input Set the vertical space around the image
IMPORTANT — Items that are grayed out will not work within this component

Image Delete

To delete an image, click the Remove the Image icon.

Bootstrap Classes

        

.media {
    margin-top: 15px;
}
.media:first-child {
    margin-top: 0;
}
.media-left, .media > .pull-left {
    padding-right: 15px;
}
.media-right, .media > .pull-right {
    padding-left: 15px;
}
.media-right img, .media > .pull-right img,
    .media-left img, .media > .pull-left img {
    margin-bottom: 15px;
}
.media-left, .media-right, .media-body {
    display: table-cell;
    vertical-align: top; /*Horizontal Align top*/
}
.media-middle {
    vertical-align: middle;
}
.media-bottom {
    vertical-align: bottom;
}
