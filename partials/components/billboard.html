Billboard

Allowed In

  • Container
  • Row
  • Row Advanced

Usage

Use a billboard to cycle through feature items, like a carousel (slideshow).


More information at getbootstrap.com
http://getbootstrap.com/javascript/#carousel
https://getbootstrap.com/examples/carousel/

Example

Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Show Dots Checkbox
Wrap Checkbox
Speed Input
Auto Play Checkbox

Add Billboard Slide


To insert an billboard slide, click the Insert a new item icon .

Allowed Components

  • Billboard Slide
