Media Object

There is a bug here - once an image is added and the page is saved, you can no longer access the Image toolbar which contains the add/modify/delete options

Allowed In

  • Accordion
  • Billboard
  • Container
  • Panel
  • Row
  • Row Advanced
  • Tabset

Usage

A starting point for individual items that will contain an image (left or right aligned) alongside text content.


More information at getboostrap.com
http://getbootstrap.com/components/#media


Additional Resources
http://www.w3schools.com/bootstrap/bootstrap_media_objects.asp

Examples

Content Group Options


To edit content group options, click the Edit Content Group icon .

Option Data Type Function
Image Media Library Maximum image width for best results is 200px
Image Link Content Selector
Media Object Text Rich Text (RTE)
Image Horz Alignment Selection Select the image horizontal alignment using default options
  • Align Left
  • Align Right
Image Vertical Alignment Selection Select the image vertical alignment using default options
  • Align Top
  • Align Middle
  • Align Bottom
CSS Class Input Add CSS class
Style Input Add CSS styles (inline)

Code Sample

<div class="media" style="">
    <div class="media-left media-top">
        <a href="...">
            <img class="media-object" src="/-/media/Images/[filename].ashx" alt="">
        </a>
    </div>
    <div class="media-body">
        ...
    </div>
</div>

Image Select

To select an image, click the Choose an Image icon.

Image Edit

To edit an image, click the Modify Image Appearance icon.


Image Properties
Option Data Type Function
Text Input Alternate Text
Width Input Set the width of the image if they are different than the dimensions of the original
Height Input Set the height of the image if they are different than the dimensions of the original
Keep Aspect Ratio Checkbox
Horizontal space Input Set the horizontal space around the image
Vertical Space Input Set the vertical space around the image

Image Delete

To delete an image, click the Remove the Image icon.

Allowed Components

None

Bootstrap Classes

        

.media {
    margin-top: 15px;
}
.media:first-child {
    margin-top: 0;
}
.media-left, .media > .pull-left {
    padding-right: 15px;
}
.media-right, .media > .pull-right {
    padding-left: 15px;
}
.media-right img, .media > .pull-right img,
    .media-left img, .media > .pull-left img {
    margin-bottom: 15px;
}
.media-left, .media-right, .media-body {
    display: table-cell;
    vertical-align: top; /*Horizontal Align top*/
}
.media-middle {
    vertical-align: middle;
}
.media-bottom {
    vertical-align: bottom;
}
