Simple Content Items

This component is comprised of a Title with optional linking and Body (RTE).

Title

Enter title as text — entry here is rendered in the h3 section.


Body

Enter body text — entry here is rendered in the child div section.
Note that this item uses the Rich Text Editor (RTE) so additional HTML can be entered.

Allowed Components

None

Code Sample

<div style="">
    <h3>Title</h3>
    <div>
        <p>Body</p>
    </div>
</div>
