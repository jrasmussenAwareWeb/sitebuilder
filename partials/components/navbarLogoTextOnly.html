Tips & Troubleshooting

If not linking text

If the text is not linked the image will be larger than if the text is linked because the .navbar-brand class is not wrapping the image.

Add additional styles below in the Custom Stylesheet as needed — these styles added for the Example in this section. 
.navbar-header {
    font-size: 1.4em;
    padding: 15px 0;
}
